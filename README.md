## How to run ?

* Have Python 2 installed
* Have MySQL / MariaDB installed (comes bundled with XAMPP stack)
* Install required python packages
`pip install -r requirements.txt`
* Run the app
`python run.py`

## Nutritional Info ?

Built with Python, Flask, MySQL, SQLAlchemy...
